DOWNLOAD WORDPRESS

Here the process of installation of WordPress


• Install XAMP or WAMP web server on the internet.
• Open the link https://wordpress.org/download/
• Create MYSQL Database
• Create new empty database with user/password
SetUp Wizard
Step 1: Extract the WordPress zip file And upload to localhost or web server
Step 2: Open browser& navigate to your WordPress file path, after seeing WordPress installer select language and click on Continue.
Step 3: You can view the database information , Click on let's go!
Step 4: Here, you have to enter MYSQL database information
1)Database Name
2)Username
3)Password
4)Database Host
5)Table Prefix
After filling click on submit button
Step 5: WordPress checks the database settings and gives confirmation Click on Run the install
Step 6: Enter administrative information.
1. Site Title
2. Username
3. Password
4. Your E-mail
5. Privacy
Click on Install WordPress button
Step 7: After installation you can see success screen You can view username and password added in WordPress. Click on Login button
Step 8: After clicking login , you will get a WordPress admin panel Enter username and password as mentioned in "Step 6" during installation and
Click on LOGIN BUTTON
