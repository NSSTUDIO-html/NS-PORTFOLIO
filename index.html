About Me

I'm a B.Sc MSCS student with a strong passion for full-stack development, AI, and interactive 3D experiences. I began my programming journey with C and Java, which helped me build a solid foundation in logic and problem-solving.

As I grew more interested in web technologies, I mastered HTML5, CSS3 (with Tailwind CSS and Bootstrap), and JavaScript, diving deep into frameworks like React.js, Three.js for 3D web experiences, and TensorFlow.js for browser-based AI.

My backend skills include Node.js (working with modules like OS, PATH, HTTP, FS), Express.js, MongoDB (Basics), and REST API development. I also understand middleware and backend routing principles.

Parallel to web dev, I’ve developed a strong command over Python and its vast ecosystem, including GUI tools (Tkinter, Turtle), data libraries (NumPy, Pandas, Matplotlib), and ML frameworks like Scikit-learn and PyTorch.

Other skills include PHP, SQL, WordPress, Tally Prime, MS Office, GitHub, and DSA basics.

I’m constantly learning and working on projects that combine web, AI, and game development to create innovative user experiences.
